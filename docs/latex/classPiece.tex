\doxysubsection{Detailed Description}
Classe abstraite représentant une pièce du jeu d\textquotesingle{}échecs. 

La classe \mbox{\hyperlink{classPiece}{Piece}} est la classe de base pour toutes les pièces du jeu d\textquotesingle{}échecs, elle est abstraite et contient des méthodes virtuelles pour définir les comportements des différentes pièces.
