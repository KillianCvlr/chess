Detailed Description

Classe abstraite représentant une pièce du jeu d'échecs.

La classe Piece est la classe de base pour toutes les pièces du jeu d'échecs, elle est abstraite et contient des méthodes virtuelles pour définir les comportements des différentes pièces.
